# Smart Construction Platform

## Overview
A full-stack construction management platform consisting of a white-label mobile application and a backend dashboard system.

This system was designed to support construction operations including project tracking, reporting, and data management across mobile and web interfaces.

---

## Architecture

### Mobile Application (Flutter)
- White-label architecture
- Supports field operations and on-site reporting
- Cross-platform (Android, iOS, Web, Desktop)

### Backend System (Django)
- REST-based backend for data and business logic
- Handles project management, reporting, and user workflows
- Includes admin dashboard and modular app structure

---

## Tech Stack

**Frontend / Mobile**
- Flutter (Dart)

**Backend**
- Django (Python)

**Other**
- REST APIs
- Gunicorn / Nginx (deployment scripts included)

---

## Features

- Project tracking and management
- Field data collection via mobile app
- Centralized backend dashboard
- Modular architecture for scalability
- White-label mobile support
